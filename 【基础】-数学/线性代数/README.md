# 目录导航

线性代数是数学的一个分支，它的研究对象是向量、矩阵、线性空间、线性变换和线性方程组等等。

线性代数中的行列式、矩阵、线性方程组、线性变换等内容广泛应用于其他数学分支。

线性代数是大学本科的一门重要的公共数学基础课程，是理科、工科、经济、管理等专业的必修课程。
